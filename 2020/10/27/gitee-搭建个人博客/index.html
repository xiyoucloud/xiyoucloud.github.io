gitee 搭建个人博客

使用 gitee + hexo 搭建个人博客

点击此处更新 pages
当前网站就是使用 gitee (码云) 搭建的静态文件网站
hexo官方教程

  1. 注册码云账号, 例: txbhandsome
  2. 创建代码仓库, 例: txbhandsome
  3. 在本地机器中搭建开发环境(npm + hexo)
  4. 修改配置文件
  5. 在码云 (gitee) 中开启 Gitee Pages 服务
  6. 访问网站, 例: https://txbhandsome.gitee.io/
  7. 增加新写的文章, 更新 gitee

使用教程

  1. hexo new “文章题目”
  2. hexo generate
  3. hexo deploy
  4. gitee -> 服务 -> Gitee Pages -> 点击更新
